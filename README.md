## 训练目标
学习和掌握K8S的实战应用，功能包括：
- 1. 什么是 Kubernetes && 搭建 Kubernetes 集群服务
- 2. 使用 Kubernetes 部署应用并实现访问
- 3. 使用 Kubernetes 实现灰度发布与滚动发布
- 4. 对 Pod 进行健康度检查
- 5. 使用 Secure 集成私有镜像库
- 6. 使用 DNS 实现 Pod 服务发现
- 7. 使用 ConfigMap 分离配置文件
- 8. 使用 污点和容忍 将镜像部署在指定的 Node
